## Course Objectives

This course is designed to support each student in the pursuit of original public health research, culminating in a citable reference made available as a preprint on [bioRxiv](https://www.biorxiv.org/) if submission for peer review is planned or [Zenodo](https://zenodo.org/) otherwise.
